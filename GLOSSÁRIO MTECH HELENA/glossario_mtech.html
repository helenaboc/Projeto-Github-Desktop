A

Algoritmo
Sequência lógica de passos usada para resolver um problema.
API
Interface que permite a comunicação entre diferentes sistemas.
Array
Estrutura de dados que armazena múltiplos valores em uma única variável.
ASCII
Padrão que representa caracteres em formato numérico.

B

Backend
Parte do sistema responsável pela lógica e processamento de dados.
Banco de Dados
Sistema organizado para armazenar e gerenciar informações.
Boolean
Tipo de dado que representa verdadeiro ou falso.
Bug
Erro no código que gera comportamento inesperado.

C

CSS
Linguagem usada para estilizar páginas HTML.
Classe
Atributo usado para aplicar estilos a múltiplos elementos.
Código Fonte
Conjunto de instruções escritas pelo programador.
Compilador
Programa que traduz código para linguagem de máquina.

D

DOM
Estrutura que representa os elementos da página como objetos.
Deploy
Processo de publicar um site ou sistema na internet.
DNS
Sistema que traduz nomes de domínio em IP.
Debug
Processo de identificar e corrigir erros.

E

Elemento
Parte de uma página HTML composta por tags.
Evento
Ação do usuário como clique ou digitação.
Escopo
Região onde uma variável pode ser acessada.

F

Frontend
Parte visual da aplicação.
Função
Bloco de código reutilizável.
Framework
Estrutura que facilita o desenvolvimento.

G

Git
Sistema de controle de versão.
GitHub
Plataforma para hospedagem de código.
Grid
Sistema de layout em grade no CSS.

H

HTML
Linguagem de marcação usada para estruturar páginas web.
HTTP
Protocolo de comunicação da web.
HTTPS
Versão segura do HTTP com criptografia.

I

ID
Identificador único de um elemento HTML.
Input
Campo de entrada de dados.
IP
Endereço único de um dispositivo na rede.

J

JavaScript
Linguagem usada para criar interatividade em páginas web.
JSON
Formato leve de troca de dados.

L

Loop
Estrutura de repetição de código.
Layout
Organização visual de elementos.

M

Mobile First
Estratégia de desenvolvimento focada em dispositivos móveis.
Margin
Espaço externo de um elemento no CSS.

N

Navegador
Programa usado para acessar sites.
Node.js
Ambiente para executar JavaScript no servidor.

O

Objeto
Estrutura que armazena dados e comportamentos.

P

Python
Linguagem de programação simples e poderosa.
Parâmetro
Valor passado para uma função.

R

Responsivo
Layout que se adapta a diferentes telas.

S

Servidor
Computador que fornece dados para outros.
String
Tipo de dado textual.

T

Tag
Elemento usado na estrutura HTML.

U

URL
Endereço de um recurso na internet.

V

Variável
Espaço usado para armazenar dados.

W

Web
Ambiente de páginas e sistemas na internet.

X

XML
Linguagem de marcação para dados.

Y

YAML
Formato de dados usado em configurações.

Z

Z-index
Define a ordem de camadas no CSS.
